# Projects

## IDS for Nanosatellite — AI/ML, Security, Research
Tags: ai-ml, security, research
Stack: Python, Random Forest, XGBoost, SVM, MLP, K-Means, NSL-KDD, STIN
GitHub: https://github.com/zeyad-hisham-cyper/Nano-Satellite-IDS
Status: Completed — journal paper in preparation
Media: diagram, plots

Designed a lightweight intrusion detection system for resource-constrained
nanosatellite communication links. The challenge was building an ML pipeline
accurate enough to be useful but lean enough to run on limited hardware.
Final architecture uses a K-Means + Random Forest hybrid model, achieving
95% accuracy under stress-test conditions. Evaluated on both the NSL-KDD
benchmark dataset and a satellite-specific traffic dataset (STIN).

## Campus Irrigation Automation — IoT, Embedded, Graduation Project
Tags: embedded, iot
Stack: ESP-IDF, ESP-NOW, MQTT, Firebase, Flutter, Solar Power
GitHub: https://github.com/zeyad-hisham-cyper/IoT-Smarrt-Irrigation-System-using-Solar-power
Status: In development
Media: pcb, hardware

Solar-powered smart irrigation system built for the MIU campus as a

## Door Locker Security System — Embedded C, UART, EEPROM
Tags: embedded, security
Stack: AVR, Embedded C, LCD, Keypad, UART, EEPROM, PIR

## AF Function Generator — Analog Electronics, PCB Design
Tags: embedded, hardware
Stack: Analog Circuits, Astable Multivibrator, Op-Amp, RC Filter, PCB
GitHub: —
Status: Completed
Media: pcb, proteus

Analog function generator producing square, triangle, and sine waveforms
from a single astable multivibrator core. The square wave feeds a
non-inverting integrator to produce the triangle wave, which passes
through an RC low-pass filter to yield a smooth sine output. Designed,
simulated in Proteus, and fabricated as a PCB prototype. Clean analog
design with no digital components.
